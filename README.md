# Kundali — Vedic Birth Chart Generator

Generate comprehensive Vedic birth charts with detailed predictions, remedies, and life guidance.

**Live app**: [vedic-kundali.onrender.com](https://vedic-kundali.onrender.com)

## What you get

- South Indian charts (D1, D9, D10, D7)
- 20+ yoga detection with strength ratings
- Lucky points (numbers, days, stone, metal, colors)
- Moon nakshatra personality reading
- Life predictions (career, marriage, wealth, health, spiritual)
- 10-year forecast tied to dasha periods
- Dosha analysis (Kaal Sarp, Manglik, Pitra, Grahan, Shrapit, Guru Chandal)
- Sade Sati timeline with phases
- Lal Kitab predictions and remedies (108 entries)
- Planet-specific remedies (gemstone, mantra, charity, fasting)
- House strength assessment (0-100 scores)
- Avkahada Chakra (for kundali matching)
- Karmic lessons (Rahu-Ketu axis, Saturn discipline)
- Personalized daily rituals

## How to use

Visit [vedic-kundali.onrender.com](https://vedic-kundali.onrender.com), enter name, date of birth, time of birth, and place of birth. Supports 1500+ Indian cities and all 30,000+ pincodes.

## Tech

Python, Flask, Swiss Ephemeris (pyswisseph), Lahiri Ayanamsa, Whole Sign houses.

## Run locally

```bash
pip install -r requirements.txt
python app.py
```

Open `http://localhost:5000` in your browser.
